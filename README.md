# AI-Powered-Recruitment-System
Developed an AI-driven recruitment system that automates candidate evaluation through a chatbot-based interview platform integrating text, audio, and emotion-aware analysis. The system combines speech-to-text processing, sentiment analysis, speech emotion recognition, and LLM-based reasoning to assess candidate responses, confidence, stress levels, and engagement during multi-stage interviews.

Built a hybrid AI architecture using NLP, machine learning, and rule-based decision logic to generate explainable and unbiased hiring recommendations. Implemented multimodal analysis pipelines for evaluating both technical and behavioral aspects of candidates, while incorporating ethical AI checks such as stress thresholds, confidence consistency, and fairness validation.

Tech Stack: Python, NLP, BERT, Speech Emotion Recognition, LLM Applications, Sentiment Analysis, Machine Learning
