# AI-Powered Recruitment System

An intelligent AI-driven recruitment platform that automates candidate interviews using NLP, speech analysis, emotion recognition, and LLM-based evaluation.

The system conducts multi-stage interviews through text and audio interactions, analyzes candidate responses in real time, and generates explainable hiring recommendations using multimodal AI pipelines.

---

## Features

- AI-powered chatbot interview system
- Speech-to-text candidate response processing
- Sentiment analysis for behavioral evaluation
- Speech Emotion Recognition (SER)
- Confidence and stress-level assessment
- Technical + HR interview evaluation
- LLM-based candidate reasoning analysis
- Explainable hiring recommendation engine
- Bias reduction and fairness validation checks
- Multi-stage interview workflow

## System Architecture

1. Candidate attends AI interview
2. Audio/Text response captured
3. Speech-to-text conversion
4. NLP + Sentiment Analysis
5. Emotion detection from voice
6. LLM-based reasoning evaluation
7. Confidence/stress assessment
8. Final candidate scoring
9. Hiring recommendation generation
